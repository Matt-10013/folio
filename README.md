# Folio

**Discover · Collect · Learn** — A personal photography discovery feed powered by Claude AI.

Folio surfaces work by different photographers in a swipeable feed. You rate photos (like/dislike), and the system learns your aesthetic preferences over time. Tap any photo to see the artist's profile with links to their portfolio.

## Tech Stack

- Vanilla JavaScript (no framework)
- Firebase Auth + Firestore
- Unsplash API
- Claude API (via Cloud Functions)
- GitHub Pages PWA
